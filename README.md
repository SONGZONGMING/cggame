# cggame
プレイヤーは小さな赤い正方形を操作し、画面内を移動する4つの大きな白い正方形を避けながら、ランダムに出現する緑の正方形に触れることを目指すゲームである。

白い正方形は画面の端に当たると、ランダムな方向と速度に変化する。また、時間が経過すると、一定の確率で白い正方形が画面外に隠れることがあり、さらに時間が進むと、隠れていた白い正方形が低い確率で突然出現し、プレイヤーを襲うことがある。

さらに、プレイヤーの赤い正方形には絵を描く機能もある。白い正方形を避けながら、同時に画面上に絵を描くことができる。
